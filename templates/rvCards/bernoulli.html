Bernoulli Random Variable

Notation: $X \sim \Ber(p)$
Parameters: $p \in [0, 1]$, the probability that $X=1$.
Description: A boolean variable which is 1 with probability $p$
Support: $x \in \{0, 1\}$
PMF: $\p(X=x) = \begin{cases} p && \text{if }x = 1\\ 1-p && \text{if }x = 0 \end{cases}$
Expectation: $\E(X) = p$
Variance: $\var(X) = p (1-p)$
